# Running the Program! 

First, user needs to install Anaconda https://www.anaconda.com/

Then


```sh
  - conda env create -f environment.yml
  or
  - conda create --name traintest --file Building_identical_conda_environment-file.txt
``` 
and 

```sh
  - conda activate traintest
``` 
finally

```sh
  - python  main.py
```
